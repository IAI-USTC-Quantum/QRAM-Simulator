Detailed Description

循环左移操作

将寄存器值循环左移指定位数，溢出的高位循环到低位

数学定义：y = (y << digit) | (y >> (N - digit))，其中N是寄存器位数

Note
Unitary性质：循环移位是双射，保证unitary
dagger操作：左移d位的dagger是右移d位（或左移N-d位）
数据类型：建议UnsignedInteger，也可用于SignedInteger
Precondition
register_1必须是激活状态
digit <= 寄存器大小（digit == size时等于恒等操作）
示例
// 4位寄存器，初始值 0b1010
ShiftLeft_InPlace("reg", 1); // 结果: 0b0101 (循环左移1位)
ShiftLeft_InPlace("reg", 2); // 结果: 0b1010 (循环左移2位)
