Detailed Description

获取中点操作（Out-of-place）

计算两个无符号整数的中点值：mid ^= (left + right) / 2

常用于量子算法中的二分查找和中点计算。

Note
Unitary性质：自伴算子，通过XOR实现
数据类型：left_id、right_id、mid_id都必须是UnsignedInteger
溢出行为：加法可能溢出，但除法后结果正确（整数除法向下取整）
Precondition
left_id、right_id、mid_id必须是UnsignedInteger类型
三个寄存器大小必须相同
所有寄存器必须是激活状态
示例
auto left = System::add_register("left", UnsignedInteger, 4);
auto right = System::add_register("right", UnsignedInteger, 4);
auto mid = System::add_register("mid", UnsignedInteger, 4);
Init_Unsafe(left, 0);
Init_Unsafe(right, 10);
// mid = (0 + 10) / 2 = 5
GetMid_UInt_UInt("left", "right", "mid");
