Detailed Description

累加乘常量操作（In-place）

实现 in-place 累加乘法：res += lhs * mult (mod 2^N) 注意：lhs寄存器不会被修改，只有res被更新。

这是一个in-place操作，需要显式实现dagger来保证unitary。 Forward: res += lhs * mult (mod 2^N) [lhs不变] Dagger: res -= lhs * mult (mod 2^N) [lhs不变]

Note
Unitary性质：lhs不变，仅res通过模加/模减更新，双射性由lhs值域保证
数据类型：lhs和res都应为UnsignedInteger
溢出行为：结果按res寄存器大小模2^N回绕
Precondition
res寄存器必须有足够位数存储结果
示例
auto lhs = System::add_register("lhs", UnsignedInteger, 4);
auto res = System::add_register("res", UnsignedInteger, 4);
Init_Unsafe(lhs, 2); // lhs = 2
Init_Unsafe(res, 3); // res = 3
// res = 3 + (2 * 4) = 11
Add_Mult_UInt_ConstUInt_InPlace("lhs", 4, "res");
