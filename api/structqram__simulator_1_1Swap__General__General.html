Detailed Description

通用交换操作（In-place）

交换两个寄存器的完整值

实现：使用std::swap交换两个寄存器的value字段。 这是一个自伴操作，应用两次等于恒等操作。

Note
Unitary性质：自伴算子，Swap^2 = I
数据类型：任意类型，但两个寄存器大小必须相同
实现细节：直接交换寄存器值，不涉及XOR或算术运算
Precondition
id1和id2大小必须相同
所有寄存器必须是激活状态
示例
auto reg1 = System::add_register("reg1", UnsignedInteger, 4);
auto reg2 = System::add_register("reg2", UnsignedInteger, 4);
Init_Unsafe(reg1, 5); // reg1 = 5
Init_Unsafe(reg2, 10); // reg2 = 10
// 交换后: reg1 = 10, reg2 = 5
Swap_General_General("reg1", "reg2");
