Detailed Description

布尔翻转操作

翻转寄存器中的所有位（按位取反），实现 y = ~y

Note
Unitary性质：自伴算子（SelfAdjointOperator），即 U^† = U
数据类型：任意整数类型（UnsignedInteger/SignedInteger）
溢出行为：只影响寄存器size范围内的位，高位被翻转但会被mask截断
Precondition
输入寄存器必须是激活状态
示例
// 4位寄存器，初始值 0b1010 (10)
FlipBools("reg"); // 结果: 0b0101 (5)
